# pdx_crime_prediction

ML crime prediction using tensorflow.js & open data

## This application is for entertainment purposes only and is not to be taken seriously.  Do not use for actual crime prediction.

### Built with:
* [Tensorflow.js](https://js.tensorflow.org/)
* [Portland Police Crime Statistics Open Dataset (January 2018-April 2018](https://www.portlandoregon.gov/police/71978)
* [Google Maps Embed API](https://developers.google.com/maps/documentation/embed/guide)
